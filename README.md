# Emotion Recognition App

A real-time emotion recognition application that detects faces from a webcam and classifies facial emotions using Computer Vision and Deep Learning.

## Features (planned)

- Real-time webcam capture
- Face detection
- Emotion recognition
- Confidence scores
- Emotion history
- Session statistics
- Docker support

## Tech Stack

- Python
- OpenCV
- NumPy
- MediaPipe
- PyTorch
